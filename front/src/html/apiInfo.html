Вы можете получить все данные сайта в виде удобной json таблицы.
Важно! Не суйте вызов api в беск цикл. Данные на сервер обновляются с задержкой (от 10 сек до 10 мин), поэтому спользуйте вызов хотябы раз в минуту!

Ссылка: sparebola.ru/virtualsamp/api?
Она принимает параметры: action и block.
"?action" = respond, players, top, gangzone, server.
"&block" = dinamic, static. Это нужно для "?action=server". Dinamic получает чат и киллстат, static всё остальное.

Примеры:
Выводит данные чата и киллстата (4 страница):
https://sparebola.ru/virtualsamp/api?action=server&block=dinamic
Выводит данные сервиса (1 страница):
https://sparebola.ru/virtualsamp/api?action=respond
Выводит данные топа (3 страница):
https://sparebola.ru/virtualsamp/api?action=top
